# jdbc-impl
순수 JDBC를 사용한 DAO 구현 (단일 조회, 연관관계 조회, CRUD, etc)
                    
## Tech Stack & Version
- Java 17
- Spring boot 2.7.11
- Spring Web
- JDBC API
- Lombok
- H2 Database
